# SIDCUP FAMILY GOLF CLONE

A responsive clone of the Sidcup Family Golf website built using HTML, CSS, JavaScript, and GSAP.

## Features

- Responsive Design
- Custom Cursor Animation
- Smooth Scrolling Effects
- GSAP Animations
- Interactive User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GSAP
- ScrollTrigger
